Pre-requirement to Learn Angular
  • HTML, DOM (Document Object Model), CSS but isn't a must-have
  • JavaScript Core
  • TypeScript
  • OOPs Concept
To install Angular pre-requiement
  • Nodejs >v16 (Angular latest version is 16)
  • NPM
  • Command to install angular cli
  • npm install -g @angular/cli
Create Angular project cmd
  • ng new nameOfProject
